1. Staff User Creation Workflow

Creating Admin, Event Director, or Judge accounts - No payment required

1
Select User Type
Navigate to /admin/members and click "Add User" button.
  • Select "Staff User" option
  • This indicates the user will have administrative roles
2
Enter Basic Information
  • Email* - Required, must be unique
  • First Name* - Required
  • Last Name* - Required
  • Phone - Optional
3
Password Setup
Choose password method:
  • Generate Password - System creates a secure random password
  • Set Manual Password - Enter custom password with strength indicator
Password must meet minimum strength requirements (shown visually).
4
Select Role
Choose the staff role:
  • Admin - Full system access
  • Event Director - Event management capabilities
  • Judge - Competition judging access
5
Review & Create
Review all information and click "Create User":
  • User account is created
  • Role is assigned
  • Welcome email is sent with credentials
  • Redirected to Member Detail page

After Staff User Creation

Staff users can optionally be assigned a membership later from their Member Detail page using the "Add Membership" button. This allows staff members to also compete in events.

2. Membership User Creation Workflow

Creating Competitor, Retailer, or Manufacturer accounts - Payment required

Part A: Create User Account

1
Select User Type
Navigate to /admin/members and click "Add User" button.
  • Select "Membership User" option
  • This indicates the user needs a paid membership
2
Enter Basic Information
Same as staff user - Email, First Name, Last Name, Phone
3
Password Setup
Same as staff user - Generate or set manual password
4
Review & Create
User is created and you're redirected to Member Detail page to assign membership.

Part B: Assign Membership (Membership Wizard)

Click "Add Membership" button on the Member Detail page

1
Select Membership Type
Choose from available membership types:
  • Competitor - Individual competitor membership
  • Retailer - Business retailer membership
  • Manufacturer Bronze/Silver/Gold - Tiered manufacturer partnerships
2
Category-Specific Information
If Competitor:
  • Competitor Name (optional - for family members)
  • Vehicle Make, Model, Color, License Plate (required)
  • Optional: Add Team Add-on (+$25/year)
If Retailer/Manufacturer:
  • Business Name (required)
  • Business Website (optional)
3
Billing Information
Complete billing details:
  • First Name, Last Name, Email, Phone
  • Address, City, State, Postal Code, Country
4
Payment Method
Select how payment will be processed (see Payment Methods section below)
5
Review & Confirm
Review all details and create the membership

When Credit Card (Invoice) is selected, the membership remains in PENDING status until the user completes payment. The member cannot compete or access member benefits until the invoice is paid.
