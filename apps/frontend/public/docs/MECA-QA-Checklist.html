MECA Website QA Testing Checklist

Systematic quality assurance testing guide for site administrators

Please test each item below and mark it as Pass, Fail, or Skip

How to Use This Checklist

  • Go section by section - start at Section 1 and work your way through
  • Follow the steps under each item to test that feature
  • Check the Expected Result to know what should happen
  • Mark each item as PASS (works correctly), FAIL (has a problem), or SKIP (not applicable)
  • Add notes for any failed items describing what went wrong
  • Sections 1-9 cover public pages and user features
  • Sections 10-20 cover admin-only features (you must be logged in as admin)
  • Section 21 tests mobile/phone display
  • Section 22 is your overall sign-off
