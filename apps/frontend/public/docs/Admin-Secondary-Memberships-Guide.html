1. System Overview

The Master/Secondary Membership system allows a single user (the "Master") to manage billing for multiple memberships. Each membership gets its own unique MECA ID, but all billing goes through the master account.

Key Benefits

  • Family Memberships: Parents can manage memberships for multiple family members
  • Team Sponsors: A sponsor can pay for multiple competitor memberships
  • Business Accounts: A business can manage memberships for employees
  • Simplified Billing: One invoice for multiple memberships

Important Concept

Each secondary membership gets its own unique MECA ID. Results and competition history are tied to the MECA ID, not the account. If a secondary later becomes independent, they keep their MECA ID and all their history.

2. Account Types Explained

INDEPENDENT

Default account type

  • Standard standalone membership
  • Handles own billing
  • Can upgrade to MASTER
  • Has own MECA ID

MASTER

Controls secondary memberships

  • Manages billing for all secondaries
  • Can add/remove secondaries
  • Can switch between MECA IDs
  • Maximum 10 secondaries (configurable)

SECONDARY

Linked to a master

  • Own unique MECA ID
  • Billing managed by master
  • Optional: own login credentials
  • Can upgrade to INDEPENDENT

Secondary Login Options

No Login
(Managed by Master)		 No separate account created. Master manages everything.
  • Compete (master registers)

  • Self-register for events

  • Register for events
  • View own results
  • View invoices/billing
  • Add memberships
  • Manage payment methods

3. Creating Secondary Memberships (Admin)

Two methods available for admins to create secondary memberships

Method A: From Admin User Wizard

/admin/members → Add User → Membership User
1
Start User Wizard
Navigate to /admin/members and click "Add User" button.
  • Select "Membership User" option
2
Enter User Information
Enter the standard user details:
  • Email, First Name, Last Name
  • Password setup (generate or manual)
3
Secondary Option Step
A new step appears asking: "Is this a secondary membership?"
  • Select "Yes" to create as secondary
  • A search field appears to find the master account
4
Search for Master Account
Search for the master membership by:
  • Email address of the master account holder
  • MECA ID of the master membership
  • Name of the master account holder
Select the correct membership from results.
5
Choose Login Option
Decide if the secondary should have their own login:
  • No login: Master manages everything
  • Own login: Secondary can log in but no billing access
6
Complete Membership Details
Continue with standard membership wizard:
  • Select membership type
  • Enter competitor name (if different from account name)
  • Vehicle information
  • Payment method (billed to master)
7
Review & Create
Review shows:
  • Master Account: [Name] (MECA ID: XXXX)
  • Secondary will receive own MECA ID upon payment
  • All billing goes to master account
Click "Create Secondary Membership" to complete.

Method B: From Member Detail Page

/admin/members/[id] → Add Secondary
1
Navigate to Master's Profile
Go to the Member Detail page of the user who will be the master:
  • Search for member in /admin/members
  • Click on their name to view detail page
2
Click "Add Secondary"
In the Memberships section, click the "Add Secondary" button next to their membership.
  • Only visible for PAID memberships
  • Master status is automatically assigned if not already
3
Complete Secondary Wizard
Fill in the secondary membership details:
  • Competitor Name
  • Vehicle Information
  • Login option (yes/no)
  • If login: Email for the secondary
  • Membership type
  • Payment method

Removing a Secondary

Important: Removal vs. Deletion

Removing a secondary does NOT delete the membership. It simply:

  • Changes their account type from SECONDARY to INDEPENDENT
  • Transfers billing responsibility to them
  • Keeps their MECA ID and competition history

5. Billing & Invoicing

How Billing Works

  • Initial Purchase: Invoice sent to master's email
  • Renewals: Each secondary has independent renewal dates
  • All invoices: Billed to master's profile and payment method

Invoice Line Items

When multiple secondaries are due at once, the invoice shows:

Invoice #INV-2025-0042 Line Items: 1. Competitor Membership - John Smith Jr. (MECA ID: 12345) $55.00 2. Competitor Membership - Jane Smith (MECA ID: 12346) $55.00 --------------- Subtotal: $110.00 Total Due: $110.00 Bill To: John Smith Sr. (Master Account)

Independent Renewal Dates

Each secondary membership has its own start and end dates. This means:

  • Secondaries created at different times renew at different times
  • Master receives separate invoices as each comes due
  • Master can choose to renew or not renew each secondary independently

When to Upgrade Secondary to Independent

  • Child turns 18 and wants own billing responsibility
  • Family member wants full account control
  • Sponsored competitor no longer sponsored
  • Upon user request
