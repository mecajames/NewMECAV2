Event Registrations

Review, edit, refund, and cancel competitor registrations

/admin/event-registrations

Audience: site administrators

The Event Registrations page lists every registered competitor across every upcoming event. Use it to spot-check a roster, fix a class assignment, or refund a registration before the event runs.

Contents

  • 1. Find a registration
  • 2. Edit a registration
  • 3. Refund or cancel a single registration
  • 4. Bulk actions across multiple registrations

1. Find a registration

1
Open /admin/event-registrations
every registration is listed with member name, MECA ID, event, classes, payment status, and check-in state.
2
Filter by event
pick from the event dropdown to narrow to one event's roster.
3
Search by member
type a name or MECA ID into the search box.
4
Click any row to open the registration detail page.

2. Edit a registration

1
Open the registration detail page.
2
Add or remove classes
adjust the class list if a member registered for the wrong class. The event's active classes are pulled from the live class list.
3
Update vehicle info
for SPL/SQL events, edit make, model, year, color and license plate.
4
Save
changes are visible on the member's My Registrations page.

3. Refund or cancel a single registration

1
Open the registration detail page.
2
Click Refund
issues a Stripe/PayPal refund for the full amount and marks the registration cancelled.
3
Click Cancel without refund
for comp registrations or where refund is not warranted; cancels access to the event without a refund.
4
Confirm
both actions are recorded in the admin audit log.

4. Bulk actions across multiple registrations

1
Tick the checkboxes next to any rows you want to act on.
2
Use the bulk action bar at the top to refund or cancel them in one shot.
3
Provide a reason of at least 5 characters
it shows on the audit entries.

Heads up

Bulk refunds cannot be reversed. Verify the selection before confirming.
