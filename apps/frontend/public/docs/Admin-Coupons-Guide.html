Coupon Codes

Create discount codes for memberships and shop checkouts

/admin/coupons

Audience: site administrators

Coupon codes drive promotional campaigns. Each coupon is scoped (memberships only, shop only, or both), can have an expiration date, and can be disabled without deleting historical usage.

Contents

  • 1. Create a coupon
  • 2. Edit or disable a coupon
  • 3. Audit redemptions

1. Create a coupon

1
Open /admin/coupons
click "+ New Coupon".
2
Set the code (e.g. SUMMER2026), discount type (percent or fixed amount), and value.
3
Set scope
Memberships, Shop, or Both.
4
Set expiration date if applicable. Leave blank for no expiration.
5
Set a usage cap if applicable (e.g. first 50 redemptions).
6
Save
the code is live and can be entered at checkout immediately.

2. Edit or disable a coupon

1
Open the coupon row.
2
Disable
sets is_active = false, blocking new redemptions without affecting historical usage.
3
Edit value or scope
changes apply to future redemptions only.

3. Audit redemptions

1
Each coupon detail page shows a list of redemptions: when, by whom, on what order.
2
Use this to validate campaign ROI or investigate disputes.
